âš™ï¸ Setup Instructions

  1. Create Last.fm Account

    Sign up at last.fm if you don't have an account

  2. Get API Credentials

    Visit last.fm/api/account/create and create an API application

  3. Add to Poke MCP Configuration

    Add this configuration to your Poke MCP settings:

    {
  "mcpServers": {
    "lastfm": {
      "url": "https://lastfm-mcp.com/mcp",
      "env": {
        "LASTFM_API_KEY": "your_api_key",
        "LASTFM_SECRET": "your_secret",
        "LASTFM_USERNAME": "your_username"
      }
    }
  }
}
  4. Restart Poke

    Close and restart Poke to load the new MCP server.

  5. Test the Connection

    Try asking Poke: "What are my top artists this week?"
