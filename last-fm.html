âš™ï¸ Setup Instructions

Follow these steps to connect Last.fm MCP to your Poke assistant:

  1. Create Last.fm Account

    Sign up at last.fm if you don't have an account.
    It's free and tracks your listening across all platforms!

  2. Get API Credentials

    1. Go to last.fm/api/account/create
    2. Create a new API application
    3. Copy your API Key and Shared Secret
    4. Save these credentials securely

  3. Install Last.fm MCP Server

    Open your terminal and run:
    npm install -g @modelcontextprotocol/server-lastfm

  4. Add to Poke MCP Configuration

    Add this to your MCP configuration file:

    {
  "mcpServers": {
    "lastfm": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-lastfm"],
      "env": {
        "LASTFM_API_KEY": "your_api_key_here",
        "LASTFM_SECRET": "your_shared_secret_here",
        "LASTFM_USERNAME": "your_username"
      }
    }
  }
}

    âš ï¸ Replace the placeholder values with your actual credentials

  5. Restart Poke

    Close and restart your Poke application to load the new MCP server.

  6. Test the Connection

    Try asking Poke: "What have I been listening to this week?"
