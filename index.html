‚ùì Frequently Asked Questions

Everything you need to know about MCP integrations

What is Model Context Protocol (MCP)? ‚ñº

Model Context Protocol (MCP) is an open standard that enables AI assistants like Poke to connect to external data sources and tools. It provides a universal way to integrate various services, databases, and APIs into your AI conversations, making them more powerful and context-aware.

How do I install an MCP integration? ‚ñº

Each integration has its own setup guide (click the "View Guide" button on any card). Generally, you'll need to: 1) Get API credentials from the service, 2) Add the MCP server configuration to your Poke settings, 3) Restart Poke to load the integration. Detailed instructions are provided for each service.

Are these integrations free to use? ‚ñº

The MCP servers themselves are open source and free. However, some services (like GitHub, Todoist, etc.) may require paid subscriptions for full access to their APIs. Most services offer free tiers that work perfectly with these integrations. Check each service's pricing page for details.

Is my data secure? ‚ñº

Yes! MCP servers run locally on your machine and communicate directly with services using your API keys. Your credentials and data never pass through third-party servers. Always store API keys securely and never share them publicly. Follow our security best practices section for more information.

Can I contribute my own integration? ‚ñº

Absolutely! This is a community-driven project. You can contribute new integrations, improvements, or documentation by visiting the GitHub repository. Create an MCP server for your favorite service and submit a pull request. We welcome all contributions!

What if an integration stops working? ‚ñº

First, check if your API keys are still valid and haven't expired. Then, verify that the service itself is operational. If the issue persists, check the integration's GitHub repository for known issues or updates. You can also report problems through GitHub issues, and the community will help troubleshoot.
