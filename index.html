ü§ñ What is MCP in Poke?

Model Context Protocol (MCP) allows Poke to connect to external data sources and tools. Once configured, you can ask Poke to access real-time information and perform actions using these integrations directly in your conversations.

Learn more about MCP ‚Üí

‚öôÔ∏è Connecting to Poke

To use these MCP servers with Poke, you'll need to add them to your Poke MCP configuration. Each integration requires specific API keys and settings. Click any integration card above for detailed instructions.

View configuration examples ‚Üí

üîë API Keys & Security

Most integrations require API keys from the respective services. Store these securely in your Poke MCP configuration. Never share your API keys publicly or commit them to version control.

üí¨ Using MCPs in Poke

Once connected, simply ask Poke naturally! For example: "Check BART delays" or "Add a task to Todoist". Click any integration card to see complete setup guides and conversation examples.

üîí Security & Best Practices for Poke Users

Secure Configuration

Store all API keys in your Poke MCP configuration file, never in plain text

Rate Limiting

MCP servers implement rate limiting to prevent API abuse and ensure fair usage

Minimal Permissions

Only grant the minimum required API permissions for each integration

Regular Updates

Keep your Poke app and MCP servers updated for the latest security patches
