Follow these steps to connect BART MCP to your Poke assistant:

  1. Install the BART MCP Server

    Open your terminal and run:
    npm install -g @modelcontextprotocol/server-bart

  2. Locate Your Poke MCP Configuration

    Find your Poke MCP configuration file. This is usually located at:
    ‚Ä¢ macOS/Linux: ~/.poke/mcp.json
    ‚Ä¢ Windows: %USERPROFILE%\.poke\mcp.json
    Or access it through Poke Settings ‚Üí MCP Servers

  3. Add BART Configuration

    Add this to your MCP configuration file:

    {
  "mcpServers": {
    "bart": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-bart"]
    }
  }
}

    ‚ÑπÔ∏è No API key required! BART provides open public data.

  4. Restart Poke

    Close and restart your Poke application to load the new MCP server.

  5. Test the Connection

    Try asking Poke: "Are there any BART delays right now?"
