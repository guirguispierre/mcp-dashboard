âš™ï¸ Setup Instructions

Follow these steps to connect FFmpeg MCP to your Poke assistant:

  1. Install FFmpeg on Your System

    macOS:
    brew install ffmpeg

    Windows:
    winget install ffmpeg
    Or download from ffmpeg.org

    Linux:
    sudo apt install ffmpeg

  2. Verify FFmpeg Installation

    Run in terminal to confirm:
    ffmpeg -version
    You should see version information.

  3. Install FFmpeg MCP Server

    Open your terminal and run:
    npm install -g @modelcontextprotocol/server-ffmpeg

  4. Add to Poke MCP Configuration

    Add this to your MCP configuration file:

    {
  "mcpServers": {
    "ffmpeg": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-ffmpeg"]
    }
  }
}
  5. Restart Poke

    Close and restart your Poke application to load the new MCP server.

  6. Test the Connection

    Try asking Poke: "Convert my video to MP4 format"
