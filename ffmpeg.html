‚öôÔ∏è Setup Instructions

Follow these steps to connect FFmpeg MCP to your Poke assistant:

  1. Install FFmpeg on Your System

    macOS: brew install ffmpeg
    Windows: winget install ffmpeg
    Linux: sudo apt install ffmpeg

  2. Verify FFmpeg Installation

    Run: ffmpeg -version

  3. Install FFmpeg MCP Server

    npm install -g @modelcontextprotocol/server-ffmpeg

  4. Add to Poke MCP Configuration 
    {
  "mcpServers": {
    "ffmpeg": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-ffmpeg"]
    }
  }
}
  5. Restart Poke

    Close and restart Poke to load the new MCP server.
